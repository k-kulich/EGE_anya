# Подготовка к ЕГЭ

Тут я буду размещать основную информацию по нашей подготовке к ЕГЭ. Как по мне, такой формат более удобный, чем на диске, так что теперь будем тусоваться тут :)

## Мини-гайд

В принципе, тут все лежит логично: inf_ege - информатика егэ, inf_ext - информатика не егэ...


Для того, чтобы загрузить к себе на комп файлики отсюда, можно либо напрямую скачать (тут есть кнопочка скачать как  архив), либо, чтобы не скачивать что-то к себе каждый раз, а только запускать одну команду в терминале, можно поставить себе гит и работать как трушный прогер. 

## Как установить к себе git и этот репозиторий

Для начала, если на компе нет гита, надо поставить его. 


Для ОС Windows нужно перейти на [сайт](https://git-scm.com/install/windows), там выбрать Standalone installer и в нем тыкнуть на верхнюю ссылку (ну, или пошарить за разницу в архитектуре процессоров и подумать, выбирать ли вторую, но первая должна работать нормально в любом случае). По ней скачается файлик-установщик, и в нем надо протыкать всякие "Принять", "Да" и т.п.


Для Mac OS нужно открыть терминал (среди приложений есть), а в нем ввести команду:

```zsh
sudo port install git
```

Возможно, он попросит ввести пароль для доступа в профиль (не в аккаунт, а в систему!), нужно его ввести, он показываться не будет (это ок).


*Все, гит установлен!*


---

Теперь нужно либо в редакторе кода открыть терминал, либо в папке, где файлы по нашим занятиям лежат, открыть командную строку (терминал) и в ней прописать (ссылка есть на гитхабе в выпадающем меню `Code`; возможно, для копирования нужно сначала будет нажать там кнопочку `Fork`, а уже потом оттуда взять свою ссылку): 

```zsh
git clone <ссылка>
```

После этого в папку добавятся все материалы, и их можно будет посмотреть у себя на компе. Теперь, чтобы свои файлы тоже сюда добавлять, нужно для начала создать отдельную ветку (я назвала homework_1, но ты можешь назвать, как душе угодно, но по-английски):

```zsh
git checkout -b homework_1
```

## Как загружать дзшку

Во-первых, надо проверить, что ты находишься в ветке с дзшкой, если нет, то создать ее:

```zsh
git checkout -b <название дз>
```

Чтобы загрузить свою домашку, тебе нужно добавить отслеживаемые файлы:

```zsh
git add <название файла с расширением, только по одному на каждую команду>
```

Потом надо сделать так, чтобы гит запомнил все твои изменения:

```zsh
git commit -m "ДЗ от такого-то числа или любая твоя подпись"
git push <ссылка>
```

Теперь на самом гитхабе нужно выбрать `Create pull request`, там выбрать свою ветку так, чтобы стрелочка вела в main, заполнить название пула и комментарий (по желанию) и отправить его (нажать на зеленую кнопку). Твой пул я увижу и смогу посмотреть со своего компа.

## Как подгрузить новую инфу после занятий

Во-первых, нужно проверить, в какой ветке ты лежишь:

```zsh
git branch
```

Если там под звездочкой выведется ветка с папкой домашки, которая еще не доделана, то надо переключиться на основную вот так:

```zsh
git checkout main
```

Если ты и так в main, то делать этого не надо.


Теперь открываешь также терминал в папке на компе и запускаешь команду:

```zsh
git fetch <ссылка>
```

И все! Все загрузится к тебе. Теперь, чтобы вернуться к недоделанной домашке, нужно переключиться на ветку с дз:

```zsh
git checkout <название ветки>
```

А если все доделано, то для новой нужно просто создать ветку:

```zsh
git checkout -b <название>
```
